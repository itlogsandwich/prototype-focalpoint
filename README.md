# Focalpoint

A system that allows individuals with nearsightedness to view projected or distant content directly through their personal devices.

This was originally a collaborative project with <a href="https://github.com/alexpqbt">@alexpqbt</a>, this was his idea that we wanted to create and scale; however he has decided to let me finish the rest of it.

I may or may not scale this further. I honestly got lazy. At the very least, I might rewrite this code entirely in rust in the future using leptos ssr.

For now, only a 1-1 share screening works locally.

## Update
